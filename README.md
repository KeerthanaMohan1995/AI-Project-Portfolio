## Business Problem
The main identifies problem of  waiting for 6-8 days for the onboarding process could be poor management with time to reduce this and stay cost effective i have bought up few issues that can be corrected to reduce the time by 50%.

1. **IT & Equipment Access Delay** — New hires don't receive email or 
   system login until day 3–5, and laptops are often ordered after the 
   start date instead of before.

2. **No Onboarding Visibility** — HR has no single dashboard showing 
   onboarding status across all new hires, so at-risk delays aren't 
   caught until they've already become a problem.

3. **Communication Gaps** — Employees have simple HR policy questions 
   (benefits, leave, dress code) but wait days for a human reply by email.

4. **Repetitive Manual Task Handling** — Welcome emails, Jira ticket 
   creation, and meeting scheduling are each done manually, one by one, 
   for every new hire.

5. **Compliance Training Gaps** — Mandatory compliance training isn't 
   automatically assigned or tracked, so completion deadlines are 
   frequently missed with no one noticing.

## Proposed Solution
An agentic AI-powered onboarding platform that automates onboarding 
tasks, tracks progress in real time, generates status reports, sends 
reminders, answers employee questions via a RAG-powered assistant, and 
gives HR management a live dashboard.

## Note on Metrics
This is a simulated case study built for portfolio purposes. Any outcome 
metrics referenced (e.g. "reduces onboarding time by 50%") are design 
targets the solution was built toward — not measured real-world results.
